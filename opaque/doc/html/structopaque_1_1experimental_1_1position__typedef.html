Detailed Description

template<typename Distance, typename O>
struct opaque::experimental::position_typedef< Distance, O >


Numeric opaque typedef to represent a position

This is a numeric typedef base class providing interoperability between two types for the + and - operations as follows:

  • position + position –> type error
  • position + distance –> position
  • distance + position –> position
  • distance + distance –> distance
  • position - position –> distance
  • position - distance –> position
  • distance - position –> type error
  • distance - distance –> distance

Template arguments for position_typedef:

  1. Distance : The distance type, which must be a numeric opaque typedef
  2. O : The opaque type, your subclass
