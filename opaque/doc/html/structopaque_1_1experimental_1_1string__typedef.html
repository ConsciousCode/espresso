Detailed Description

template<typename S, typename R>
struct opaque::experimental::string_typedef< S, R >


String opaque typedef base type

This is an opaque typedef base class for standard strings.

Template arguments:

  1. S : The string type (e.g. std::string)
  2. R : The result type, your subclass
